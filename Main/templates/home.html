자동진단

서버의 취약점 진단을 위해 Script를 활용하여 진단을 진행합니다.

OS 환경에 맞게 필요한 Script 파일을 다운로드 후 관리자 권한으로 실행하면

진단 결과를 확인 가능합니다. 실행 결과 나오는 good, bad, action, log 디렉터리를 통해

양호, 취약한 항목, 조치방법, 현황을 확인할 수 있습니다.

● 진단 가능 OS - Windows - Windows Server 2012 R2 기반
● Linux - Cent OS 7 기반

수동 진단 수동 진단에서 사용한 Script와 동일한 Script로 진단을 진행하기에 신뢰하실 수 있습니다.

SSH 연결을 통한 원격 실행을 위해 관리자 권한을 필요로 합니다.

안전한 진단을 위해 관리자 계정의 비밀번호를 변경 후 진단, 다시 변경할 것을 권장합니다.

진단 결과를 웹을 통해서 편리하게 한눈에 확인 가능하며, pdf 보고서로 다운로드 가능,
이전 기록들과 비교 확인 등의 서비스를 제공합니다.
